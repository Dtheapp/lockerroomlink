## Theme Rules (UPDATED December 2025)

### Light Mode = Purple
- Primary: `#9333ea` (purple-600)
- Hover: `#7c3aed` (purple-700) 
- Light: `#a855f7` (purple-500)
- Text: `#9333ea`

### Dark Mode = Orange
- Primary: `#ea580c` (orange-600)
- Hover: `#f97316` (orange-500)
- Light: `#fb923c` (orange-400)
- Text: `#f97316`

### Implementation
CSS overrides in `styles/osys-design-system.css` automatically convert `orange-*` classes to purple in light mode.
